# Despliegue-de-Infraestructura-Multi-Contenedor-con-Docker
Proyecto 3er parcial

# Objetivo General

Diseñar, contenerizar y desplegar una aplicación completa utilizando:
- Docker
- Docker Compose
- Dockerfiles
- Redes entre contenedores
- Variables de entorno
- Persistencia mediante volúmenes

# Introduccion

El presente proyecto correspondiente al tercer parcial tiene como finalidad integrar y aplicar los conocimientos adquiridos a lo largo del curso mediante el desarrollo de una propuesta estructurada que permita analizar una problemática específica y plantear una posible solución, através de este trabajo se busca fortalecer habilidades como la investigación, el pensamiento crítico, el trabajo en equipo y el uso de herramientas tecnológicas, elementos fundamentales en la formación académica y profesional.
Durante el desarrollo del proyecto se llevaron a cabo distintas etapas que incluyeron la identificación del problema, la recopilación de información relevante, el análisis de datos y la elaboración de una propuesta que responde a las necesidades detectadas. De esta manera, se promueve un aprendizaje significativo basado en la práctica, donde la teoría se complementa con su aplicación en contextos reales.
